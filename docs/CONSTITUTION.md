# Constitution

## What This Is

A minimal axiomatic model of execution structure, separating execution from observation without introducing normativity.

## Governing Principles

1. **Structural only.** No policy, governance, or domain semantics.
2. **Observation-free projections.** Canonical projections exclude observational artifacts.
3. **Total order.** The execution stream is append-only and totally ordered.
4. **Replay equivalence.** Same inputs, same projection.

## What This Is Not

- Not a governance model.
- Not a policy framework.
- Not a runtime system.
